VeggieMarket

### CustomerModel (廃棄済み)

### Customerプロジェクト
* 認証サーバー
* 顧客データベース等のORM(Object Relational Mapper)

* Node.js+RabbitMQのためのAMQPアダプター
* 元々はBlog系のアプリとして設計・実装したが、フロントエンドとして再設計中
* 予定：coffeescript+Sass+Jade+Gruntの追加を予定、技術面では練習作があるので付け加える

### integrationプロジェクト
* EAI/EIP サーバー
* Spring Integrationサーバー
